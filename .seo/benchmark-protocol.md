## Benchmark Protocol
- Queries: .seo/queries.txt (fixed list — add only with user approval)
- Search engines: Google, Bing (locale: en-DE / en-US)
- AI answer engines: Perplexity, ChatGPT browse
- For each query record: ranking URL (if any), cited sources, answer snippet quality, our page cited Y/N
- Same query wording every run
